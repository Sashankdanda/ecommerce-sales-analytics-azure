# Cloud-Based E-Commerce Sales Analytics Platform

## 📌 Project Overview
This project is a cloud-based analytics system built using Microsoft Azure and Power BI to analyze e-commerce sales data.

## ⚙️ Technologies Used
- Azure Blob Storage
- Azure SQL Database
- Power BI
- SQL

## 📊 Features
- Sales analytics dashboard
- Product performance analysis
- Customer insights
- Data visualization

## 🔄 Workflow
CSV → Azure Blob Storage → Azure SQL → Power BI

## 📁 Dataset
Includes customers, products, and orders data.

## 📸 Screenshots
Included in screenshots folder.

## 📈 Results
Interactive dashboards showing revenue, product trends, and analytics.
